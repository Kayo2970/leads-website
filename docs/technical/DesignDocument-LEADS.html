04 — Inspiration Analysis

What We Borrowed & Why

5 sites analyzed. Patterns extracted and mapped to specific LEADS pages and components.

Xolio xolio.io

  • Branded preloader — spinning dual-colour ring fades out on DOMContentLoaded
  • Oversized hero headline — 900 weight, -1px tracking, 52px+, single bold statement
  • Staggered scroll reveal — each element animates in with 100ms sibling offset
  • Centred hero layout with clear single-focus CTA
  • Scroll parallax — background layer moves at 0.5× scroll speed vs. foreground
Applied to: All pages (preloader, scroll reveal), Home & BLS (hero style, parallax)

Pixlab pixlab.io

  • Geometric shape overlays in hero — rotated rectangles, circles as floating decorations
  • Multi-layer parallax — 3 depth layers moving at different CSS transform rates
  • Card grid with thumbnail, category tag overlay, and hover reveal effect
  • Multi-level dropdown nav with animated reveal on hover
Applied to: Hero (floating shapes), Events gallery cards (hover overlays), Navigation

Oifolio oifolio.com

  • Masonry grid — CSS column-count: 3 + break-inside: avoid
  • Category filter tabs re-sort gallery without page reload
  • SVG decorative shapes (blobs, rings) as non-interactive section accents
  • Progress bars animate width from 0 to target on scroll entry
  • Modal video player triggered by card click
Applied to: Events (masonry, filter, modal), About & Join Us (progress bars, SVG accents)

Adelux adelux.com

  • Social proof avatar cluster — stacked overlapping circles + member count copy
  • 4-column icon service grid — gradient icon box, title, 2-line description
  • Testimonial carousel — auto-advances every 5s, dot nav, CSS translateX slide
  • Community-first hero messaging: belonging, growth, real outcomes
Applied to: Home (avatar cluster, icon cards), BLS & About (testimonial carousel)

Ventix ventix.com

  • Dual dark/light theme — smooth 300ms CSS custom property transition on toggle
  • Section fade-up transitions — opacity + translateY on IntersectionObserver entry
  • Icon-paired feature lists — SVG/emoji left, label + 1-line desc right, 2-col grid
  • 9 demo layout variants confirming pattern flexibility across pages
Applied to: Entire site (dark mode), All pages (scroll reveal), Join Us & About (icon lists)

10 — Dark Mode

Dual Theme System

300ms smooth transition. Persists in localStorage. Inspired by Ventix's seamless dual-theme approach.

☀️ Light Mode

BG: #F0F7FF
Surface: #E8F1FC
Card: #FFFFFF
Text: #1A1A2E
Text 2: #4A5568

🌙 Dark Mode

BG: #0A1628
Surface: #0F2040
Card: #142050
Text: #F0F0F0
Text 2: #A0A0A0

Implementation Pattern

// Toggle
body.classList.toggle('dark');
localStorage.setItem('theme', body.classList.contains('dark') ? 'dark' : 'light');

// On load — before first paint to prevent flash
if (localStorage.getItem('theme') === 'dark') body.classList.add('dark');

// CSS — tokens swap via .dark class
body       { --bg: #F0F7FF; --card: #FFFFFF; --text: #1A1A2E; }
body.dark  { --bg: #0A1628; --card: #142050; --text: #F0F0F0; }

// All surfaces inherit via var() — zero duplication
Ventix pattern applied: transition: 0.3s ease on background, color, border-color, and box-shadow on body and all major surface elements. Theme switch feels seamless, not jarring.
